
## 5. ヒーロービジュアル ディレクション

### コンセプト
「4つの精神機能の交点」を抽象的に表現。数式でも人物でもなく、**エネルギーの収束点**として表す。

### 実装（CSS のみ）
```
外枠: 正方形コンテナ（320×320px / mobile: 260×260px）
球体: position:absolute の circle × 4
  サイズ: 各 180px × 180px（重複 40px）
  配置: 上左（W）/ 上右（E）/ 下左（L）/ 下右（F）
  スタイル: border-radius:50%; filter:blur(40px); opacity:0.65;
  
中心点: 絶対配置の白色光点（20px, opacity:0.8, blur:4px）

アニメーション: 4球体が緩やかに拡縮（scale 1→1.08→1 / 6s alternate infinite）
  各球体に animation-delay をずらして有機的な動きに
  prefers-reduced-motion: none ですべて停止
```

```

  ← ロゴ                    X/32
  [━━━━━━━━━━━━░░░░░░░░] プログレスバー（軸カラーで塗る）

[メインエリア]
  ┌──────────────────────────────┐
  │  ⬤ （軸インジケーター：小さな色点）      │
  │                              │
  │  設問テキスト（Noto Serif JP）        │
  │                              │
  │  ────────────────────────────  │
  │                              │
  │  1  2  3  4  5  6  7        │
  │  ○  ○  ○  ○  ○  ○  ○        │
  │                              │
  │  全くあてはまらない  非常にあてはまる  │
  │        どちらでもない               │
  └──────────────────────────────┘

  ← 前の質問
```

### スケールボタン仕様
```
通常: 直径42px, bg: --bg-card, border: --border
ホバー: border-color: 軸カラー, scale(1.05)
選択: bg: 軸カラー, border: 軸カラー, shadow: 軸カラーglow
フォーカス: outline: 2px solid --brand（WCAG AA）
```

### 遷移アニメーション
```
設問切り替え: slide-up (translateY 20px→0 + opacity 0→1 / 250ms ease-out)
回答後自動進行: 300ms delay → slide-up 次の設問
prefers-reduced-motion: fadeOnly (opacity only, no transform)
```
